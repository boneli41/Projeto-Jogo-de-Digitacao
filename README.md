# Sistema de Ensino de Digitação para Idosos

Este projeto foi desenvolvido como parte de um trabalho acadêmico e tem como objetivo auxiliar pessoas idosas no aprendizado de digitação em computadores.

A aplicação utiliza conceitos de gamificação e uma interface intuitiva para incentivar a prática contínua, tornando o processo de aprendizagem mais acessível, motivador e eficiente.

## Funcionalidades
- Exercícios de digitação interativos
- Feedback em tempo real
- Interface simples e amigável
- Progressão baseada em desempenho

## Tecnologias Utilizadas
- Java
- JavaFX (ainda não implementado)

## Objetivo
Promover a inclusão digital de pessoas idosas, facilitando o uso de computadores por meio do desenvolvimento da habilidade de digitação.
